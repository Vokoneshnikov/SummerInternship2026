# API интернет-магазина

### REST API с каталогом товаров с синхронизацией цен с внешним прайс-листом


## Подробнее

1. Есть некий JSON документ, в котором находятся актуальные прайсы для каждого товара - его заполняет сторонее лицо (по сути API, но не совсем).
2. Есть БД, в которой хранится продублированная JSON документа информация - ею мы будем пользоваться при расчетах цен и т.п.
3. ВАЖНОЕ УТОЧНЕНИЕ: JSON документ - внешний прайс лист, который используется для актуализации цен, а БД - инструмент для API, с которым мы взаимодействуем

## Для реализации проекта необходимо:

### 1. Использовать внешний API для конвертации валют

В своем проекте я использую следующий API: https://api.exchangerate.fun


Основной запрос выглядит следующим образом:

```

https://api.exchangerate.fun/latest?base={ВАЛЮТА}

```

Тело ответа(JSON):


Раз в час приложение делает запрос к API для каждой из валют и записывает текущие курсы в таблицу БД с тремя значениями:

1. Базовая валюта
2. Валюта, в которую конвертируют
3. Курс
